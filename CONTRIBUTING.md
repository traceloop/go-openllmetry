### Prerequisites

- `go` version `v1.21.0+`

## Development

Clone the repository:

```bash
git clone https://github.com/traceloop/go-openllmetry
cd go-openllmetry
```

## Release

```bash
git tag -a <module>/v<sem version> -m "release: <module>@v<sem version>"
```

For example:

```bash
git tag -a traceloop-sdk/v0.0.4 -m "release: traceloop-sdk@v0.0.4"
```
